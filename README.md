# Анализатор частоты слов в тексте

Это веб-приложение на FastAPI, предназначенное для анализа текстовых файлов с подсчетом частоты слов и генерации отчетов в формате Excel.

## Основные возможности

- Загрузка текстовых файлов (.txt)
- Подсчет частоты слов с нормализацией форм (лемматизация)
- Подсчет вхождений слов по строкам текста
- Генерация отчета в формате XLSX с детализацией
- Поддержка русского языка
- Ограничение на одновременные обработки для предотвращения перегрузки

## Архитектура проекта

```
words_reader/
├── main.py                # Точка входа и инициализация FastAPI приложения
├── public_report_export.py # Роутер для обработки запросов экспорта отчетов
├── word_frequency.py      # Класс для обработки текста и генерации отчетов
├── requirements.txt       # Зависимости проекта
├── words.txt              # Файл со списком слов (используется для чего-то)
├── uploads/              # Временные файлы загрузок
└── outputs/              # Сгенерированные отчеты
```

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python main.py
```

Приложение будет доступно по адресу `http://127.0.0.1:8000`

## API Endpoint

### `POST /public/report/export`

Загружает текстовый файл и возвращает отчет в формате Excel.

**Параметры:**
- `file`: Текстовый файл (.txt) для анализа

**Ответ:**
- Файл Excel с отчетом о частоте слов

**Формат отчета:**
- Словоформа (нормализованная форма слова)
- Общее количество вхождений
- Количество вхождений по строкам (через запятую)

## Зависимости

Основные зависимости:
- `fastapi` - веб-фреймворк
- `uvicorn` - ASGI сервер
- `pymorphy3` - библиотека для морфологического анализа русского языка
- `openpyxl` - работа с Excel файлами
- `aiofiles` - асинхронная работа с файлами

## Принцип работы

1. Пользователь загружает текстовый файл через API
2. Файл сохраняется во временную папку `uploads/`
3. Система обрабатывает текст, разбивая его на слова и нормализуя формы с помощью pymorphy3
4. Подсчитывается частота каждого слова и его вхождения по строкам
5. Генерируется Excel отчет с результатами
6. Временный файл удаляется
7. Пользователю возвращается сгенерированный отчет

## Безопасность и ограничения

- Разрешены только файлы с расширением .txt
- Максимальное количество одновременных обработок: 3
- Временные файлы автоматически удаляются после обработки
- Поддержка только UTF-8 кодировки
